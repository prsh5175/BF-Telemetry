BF Telemetry sound files

Place this folder on your radio SD card at:
/WIDGETS/BFTelem/sounds/

Expected file names (mono WAV, 16-bit PCM, 32 kHz or similar):
- armed.wav
- disarmed.wav
- batlow.wav
- batcrit.wav
- lowrssi.wav

If a file is missing, BF Telemetry will try /SOUNDS/en/<name>.wav as a fallback.
